# 🚀 Petite Fusée

Jeu calme et éducatif pour enfants dès 3 ans.

- Suivre la fusée avec la souris / trackpad
- Appuyer sur espace pour un petit coup de pouce lumineux
- Musique douce et voyageuse
- Pas de clic requis
- Couleurs apaisantes, sans lumière bleue vive

## Développement

```bash
pnpm install
pnpm dev
```

## Déploiement

Pousser sur `main` → Vercel déploie automatiquement.
